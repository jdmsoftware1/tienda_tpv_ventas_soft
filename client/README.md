# Frontend - Sistema de Gestión de Tienda

Frontend desarrollado con React + Vite + TypeScript + TailwindCSS.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El frontend se ejecutará en `http://localhost:5173` y se conectará al backend en `http://localhost:3000`.

## Build para Producción

```bash
npm run build
```

Los archivos compilados se generarán en `../public` para ser servidos por NestJS.

## Tecnologías

- **React 18** - Framework UI
- **Vite** - Build tool
- **TypeScript** - Tipado estático
- **TailwindCSS** - Estilos
- **React Router** - Navegación
- **Zustand** - State management
- **Axios** - HTTP client
- **Lucide React** - Iconos
- **React Hot Toast** - Notificaciones
